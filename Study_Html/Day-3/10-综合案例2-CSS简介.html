CSS（层叠样式表）

层叠样式表[Cascading Style Sheets,编写为CSS)，是一种 样式表 语言，用来描述 HTML或 XML(包括如 SVG、MathML、XHTML之类的 XML 分支语言)文档的呈现。CSS 描述了在屏幕、纸质、音频等其它媒体上的元素应该如何被染的问题。

CSS 是开放网络的核心语言之一，由 W3C规范 实现跨测览器的标准化。C5S节管了大量的工作。样式可以通过走义保存在外部,c5 文件中，同时控制多个网页的布局，这意味着开发者不必经历在所有网页上编辑布局的麻烦，CSS 被分为不同等级:CSS1现已废弃，CSS2.1是推得标准，CSS3 分成多个小模块目正在标准化中。
